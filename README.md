# Skinny to Beast Clicker

Unity mobile clicker game about transforming a skinny beginner into a powerful gym beast.

## Core idea

The player taps to train. Each tap gives training progress, strength, coins, and visual body transformation.

**Main fantasy:** from skinny beginner to gym legend.

## MVP scope

- One main mobile screen
- Tap-to-train core loop
- Strength and coin progression
- Upgrade system
- Body stage progression
- Basic UI counters
- Android-first build target
- Monetization planned later: rewarded ads, no-ads purchase, boosters, cosmetic skins

## First gameplay loop

1. Tap the character or training button.
2. Gain reps and strength.
3. Earn coins after progress milestones.
4. Buy upgrades: dumbbells, protein, better gym, coach, auto-training.
5. Unlock new body stages.

## Body stages

1. Skinny
2. Beginner
3. Fit
4. Athletic
5. Big
6. Beast
7. Gym Legend

## Tech direction

- Engine: Unity 2D
- Platform priority: Android / Google Play
- Secondary platform: iOS / App Store later
- Architecture: simple modular scripts for player stats, tapping, upgrades, economy, and UI

## Development rule

Start small. Do not build a huge RPG first. The first target is a playable MVP that can be tested on a phone.
