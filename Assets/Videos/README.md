# Videos

Put the generated main menu loop here:

```text
Assets/Videos/MainMenuLoop.mp4
```

Expected current video settings:

- Format: MP4 / H.264
- Duration: about 10 seconds
- Resolution: 1280x720
- Usage: animated main menu background

After copying the video into this folder, open Unity and run:

```text
Tools > Skinny To Beast > Create Video Main Menu Scene
```

This creates:

```text
Assets/Scenes/MainMenu.unity
```
